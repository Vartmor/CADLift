Use the CADLift API in your applications:

Create Job

POST /api/v1/jobs
Content-Type: multipart/form-data

job_type: "cad" | "image" | "prompt"
mode: "cad" | "image" | "prompt"
upload:  (for cad/image modes)
params: {"extrude_height": 3000, "wall_thickness": 200}

Check Job Status

GET /api/v1/jobs/{job_id}

Response:
{
  "id": "...",
  "status": "queued" | "processing" | "completed" | "failed",
  "output_file_id": "...",
  "metadata": {...}
}

Download Output

GET /api/v1/files/{file_id}?format=step

Supported formats:
- step, dxf, obj, stl, ply, gltf, glb
