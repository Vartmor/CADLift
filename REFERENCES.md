# References & Acknowledgments

CADLift builds upon excellent open-source projects. While not bundled directly, these projects inspired or are used via external installation:

## External Tools (Install Separately)

| Tool | Purpose | Installation |
|------|---------|--------------|
| [Online3DViewer](https://github.com/kovacsv/Online3DViewer) | WebGL 3D viewer | `npm install online-3d-viewer` |
| [Mayo](https://github.com/fougue/mayo) | Professional CAD converter | [Download releases](https://github.com/fougue/mayo/releases) |
| [OpenSCAD](https://github.com/openscad/openscad) | CAD rendering engine | [openscad.org](https://openscad.org/downloads.html) |
| [Stable Diffusion](https://github.com/CompVis/stable-diffusion) | Image generation | Via HuggingFace `diffusers` |

## AI Models (Downloaded on First Use)

- **TripoSR** weights from HuggingFace: `stabilityai/TripoSR`
- **Stable Diffusion** model: `runwayml/stable-diffusion-v1-5`
